# Pizza Sales Performance Analysis Dashboard 🍕📊

## Project Overview
This project focuses on analyzing pizza sales data using Power BI to uncover customer ordering behavior, sales trends, and operational insights.

The dashboard was designed to help understand:
- Revenue performance
- Peak ordering hours
- Best-selling pizza categories
- Daily sales trends
- Customer purchasing patterns

---

## Dataset Highlights
- 21K+ customer orders analyzed
- 50K+ pizzas sold
- $817K+ total revenue generated

---

## Key Business Insights
- Nearly 45% of orders were placed during lunch and evening rush hours
- Classic pizzas contributed ~27% of total revenue
- Friday recorded the highest order volume
- Large-size pizzas generated a significant share of sales revenue
